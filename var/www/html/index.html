TM129 RobotLab Environment

RobotLab

To run RobotLab, launch a Linux Desktop Session and then click on the RobotLab icon in the tray at the top of the desktop screen. The Neural and Remote can also be launched from there.

If you need to launch the application from a command line, open a terminal and run the command under Wine:

  • wine /opt/Apps/RobotLab/RobotLab.exe
  • wine /opt/Apps/Neural/neural.exe
  • wine /opt/Apps/Remote/Remote.exe
